# Test work


```
Install dependency

npm install

You need to create a .env file at the root of the project and
following the example provided in the .env-example file, create fields with variables.

In the application, there are three pages. The first page is the Login page, the second page is the 
Registration page, and the third page is the Home page, where the list of application users is displayed. 
Redirect to the Home page occurs after authorization.
```
